Installation

After unzipping the files the group id in "groupid.txt"  must be changed to the one you want to use.
The Instance range in "instRange.txt" must be changed to your texture instance range assigned by the BSC Texture Index,"instRange.txt" uses the same format as Sc4Tool.

Adding Files

The simplest way to create a batch list is to drag and drop a group or folder of images onto the list, this method will always create a new list.
The "Add" button can also be used to add images to the list  by either clicking on the button and browsing for the images or dragging and dropping  a group or folder of images onto it.

For the program to find the alpha map it must be named using the format "filename_a.bmp".

If the image is a png that contains transparency or an alpha map is found it will be a Dxt3 fsh if the file name begins with "hd" and it is at least 256x256 it will be a 32-bit hd fsh.

Or if the image does  not contain transparency or an alpha map is not found an white alpha map will be generated and it will be a Dxt1 fsh if the file name begins with "hd" and it is at least 256x256 it will be a 24-bit hd base texture fsh.
In the "Add" function the Group id in the Group box overrides the one in "groupid.txt"

If the name of the image starts with 0x it will use the name as the instance id.
Opening a group or folder of images with the program will add the images to a new list.

Removing files

A file may be removed by selecting it in the list and clicking the  "Remove" button.
To clear the entire list click the "Clear list" button.

Changing File options



When an image is selected in the list it's Group and instance ids  and the type of fsh can be changed.
The End format  radio buttons select the end digit of the instance number.

Processing

The "Automatically process Mips" function  generates Mipmaps for the different zoom levels.
The "Compress dat" function compresses the items when inserting them into the dat.

The "Fshwrite compression" option saves the DXT1 and DXT3 fsh files using a higher quality compression.

If the "Process" button is used the "Output folder" button changes the folder to save the processed files, the program will output the files and their mipmaps along with the .TGI files for The Reader to import.

The other way to process the images is to use the "Save dat" button this processes the images and saves them into a new dat.

Command Line Arguments

The program's Command line arguments are:

PngtoFshBatch.exe  images [/outdir:<directory>] [/group: <groupid>] [/mips] [/dat:<filename>] [/proc] [/?]

the parameters are:

images a list or folder of images to process separated by spaces

/outdir:<directory> output the fsh files from /proc into directory.

/group: <groupid> Assign the <groupid> to the files, this parameter overrides

/mips generate mipmaps for the zoom levels

/dat:<filename> process images and save them as a dat

/proc process images and save

/? show the help

Paths containing spaces must be encased in quotes
Parameters in brackets are optional.

for example:


The specified arguments would take all the images in "C\sc4bmps\New Folder" and assign them the group id of  1ABE787D, generate mipmaps for the zoom levels and then save them all into test.dat.
